# 記帳本
記錄您的生活開支，幫您分項計算家居物業、交通出行、休閒娛樂、餐飲食品、其他等類別的支出總金額。

## 功能
- 新增帳目
- 修改、刪除帳目
- 統計開銷總金額
- 依照類別篩選開銷項目

### 使用工具
- mongodb 4.2.9
- mongoose 5.10.5
- nodemon 2.0.4
- Node.js 10.22.0
- Express 4.17.1
- Express-handlebars 5.1.0
- body-parser 1.19.0
- method-override 3.0.0

### 安裝與使用方法

- 下載專案
`https://github.com/chienan/expense-tracker.git`

- 安裝Node Package Manager
`npm install`

- 新增種子資料
`npm run seed`

- 使用nodemon啟動伺服器／正常啟動
`npm run dev / npm start`
